How to do the Home Teaching Report

Must be turned in on the 1st Thursday after the 1st Tuesday of the month

District Leaders (or Home Teaching District Supervisor) should make calls and enter Home Teaching before starting the report.

To fill out the Home Teaching Report, you will use the Attendance Roll along with information you will get from lds.org.

First, make a copy of the Home Teaching Report (a Microsoft Excel file provided by the stake). Alternatively you could use the Web version I created here:
https://ironjb.github.io/lds/ht-report/

Enter the previous month (or the month you are doing the report on)

Enter the number of Elders

  • Go to LDS.org
  • In the upper right click My Account and Ward
  • Click on Leader and Clerk Resources (sign in if you need to)
  • Click on the Organization dropdown
  • Click on Elders Quorum
  • Click on the Members tab
  • Click on the Options dropdown
  • Under Filter By, select Elder

Enter the number of Prospective Elders

  • Same as getting the number of Elders like you did in the previous step, except you Filter By: Prospective Elder instead.

Enter the number of Elders that attended

  • Count the number from the Attendance Role (Not including the Prospective Elders. Also, Don't forget the Elders from Primary, YM, etc...)

Enter the number of Prospective Elders that attended

  • Count the number from the Attendance Roll

Enter the number of Assigned Families Home Teachers

  • Go to Organizations > Home and Visiting Teaching.
  • Select the Overview tab.
  • Click the Totals button on the right side (as opposed to Percentages).
  • To the right of Elder Visits you will see numbers like XX/XX. Use the number on the right (XX/XX).

Enter the number of Families that were visited

  • This time use the number on the left (XX/XX).

Enter the number of Families Not Assigned

  • On the Overview tab, use the number to the right of Households Not Assigned to be Home Taught

Enter the number of Companionships

  • Go to the Companionships tab.
  • Just below each District it shows the number of Companionships for that District.
  • Get this number for each District and add them up.

Enter the number of PPIs the EQ Presidency did for the month

  • Get this number from each District Leader and add them.

List the Elders not attending meetings

  • Does not need to be ALL the Elders not attending Elders Quorum. Just the ones who sometimes or normally come, but have not for the month you are reporting on.

List the names of the Prospective Elders working toward ordination

  • This is the list of Prospective Elders the EQ Presidency is focusing on.

For the Families not visited by Home Teachers

  • Just put see attached list (you will attach another file to the email you will send out)
  • To download the PDF of that list, click on Organizations > Home and Visiting Teaching.
  • Make sure the Overview tab is selected.
  • Make sure the Percentages is selected. (On the right side of the page.)
  • Click Print. (A dropdown will appear.)
  • Check Statistics by Month and Households Not visited in 3 Months
  • Click the secondary Print button. (The button below the checkboxes you just checked.)
  • The PDF will open in a separate browser tab or window. Make sure to save it so you can attach it to the email later.

As for the Membership Record Maintenance, I never really did anything with this since it is in LDS Tools, but if they start asking for it then enter in that info as needed.

Now email the HT Report along with the PDF you downloaded for Statistics by Month and Households Not visited in 3 Months

You should email the Bishop, EQ President and Bro. Given (Stake High Councilor assigned to our Ward).
